# C13(13-role) 역할별 세부 행동 선택 규칙

## 1) 공통 세부 행동 목록 (모든 역할 공통 UI)

아래 항목은 **모든 C13 역할에 공통으로 노출**되는 유저 선택 항목이다.
각 항목은 매치엔진의 실제 액션/결과(shot_diet/participants)와 직접 연결된다.

### 온볼(On-ball)

| 코드 | 항목(유저 선택) | 의미(많이/적게) | 매치엔진 연결 액션 |
|---|---|---|---|
| `ON_PULLUP` | 풀업 점퍼 시도 | 드리블 점퍼(3점/미드) 적극성 | `ISO/PnR`의 `SHOT_3_OD`, `SHOT_MID_PU` |
| `ON_RIM_ATTACK` | 림 진입/돌파 | 드라이브로 림 압박, 림 마무리 진입 | `Drive` + `SHOT_RIM_*`, `FOUL_DRAW_RIM` |
| `ON_PNR_BH` | PnR 볼핸들러 수행 | 스크린 사용 후 의사결정 주도 | `PnR`, `PnP`에서 볼핸들러 가중 |
| `ON_POST_TOUCH` | 포스트업 터치 | 포스트 기점으로 플레이 시작/마무리 | `PostUp` + `POST_*` 관련 결과 |
| `ON_PASS_FIRST` | 패스 우선(0.5) | 킥아웃/엑스트라패스 우선 | `PASS_KICKOUT`, `PASS_EXTRA`, `RESET_HUB` |
| `onball_isolation` | 아이솔레이션 | 1:1 고립 상황에서 직접 해결 비중 | `ISO` 액션 비중 및 `SHOT_3_OD`, `SHOT_MID_PU`, `SHOT_RIM_*` 분기 |

### 오프볼(Off-ball)

| 코드 | 항목(유저 선택) | 의미(많이/적게) | 매치엔진 연결 액션 |
|---|---|---|---|
| `OFF_SPOTUP_SPACE` | 스팟업/간격 유지 | 코너/윙 정지 스페이싱, C&S 준비 | `SpotUp`, `SHOT_3_CS`, `SHOT_MID_CS` |
| `OFF_CUT` | 컷/백도어 | 헬프 뒤 침투 및 림 마무리 연결 | `Cut`, `SHOT_RIM_*`, `FOUL_DRAW_RIM` |
| `OFF_OFFSCREEN` | 오프스크린 활용 | 핀다운/핸드오프를 통해 슈팅 기회 창출 | `DHO`, (오프스크린 이후) `SHOT_3_CS/OD` |
| `OFF_SCREEN_ROLE` | 스크리너 성향(롤↔팝) | 스크린 이후 림 롤/외곽 팝 선택 성향 | `PnR`의 롤/`PnP`의 팝 분기 |
| `offball_crash_off_reb` | 공격 리바운드 가담 | 슛 이후 세컨드 찬스 확보를 위한 림 어택/박스아웃 경합 참여 | `REB_OR` 기반 공격리바운드 참여도 및 세컨드 찬스 포제션 |

## 2) 역할별 "선택 불가(잠금)" / "선택 가능" 규칙

해석 원칙:
- **선택 불가(잠금)**: 역할 정의만으로 방향성이 명확한 항목(예: SpotUp_Spacer의 스팟업 많이).
- **선택 가능**: 역할명/현재 엔진 우선순위만으로 단정하기 어려운 항목.

표기:
- `잠금(많이)` / `잠금(적게)` = 유저 조정 불가
- `선택 가능` = 유저 조정 가능

## 3) 구현 시 권장 데이터 구조

```json
{
  "behavior_catalog": {
    "on_ball": ["ON_PULLUP", "ON_RIM_ATTACK", "ON_PNR_BH", "ON_POST_TOUCH", "ON_PASS_FIRST", "onball_isolation"],
    "off_ball": ["OFF_SPOTUP_SPACE", "OFF_CUT", "OFF_OFFSCREEN", "OFF_SCREEN_ROLE", "offball_crash_off_reb"]
  },
  "role_behavior_rules": {
    "Engine_Primary": {
      "locked": {"ON_PNR_BH": "HIGH"},
      "selectable": ["ON_PULLUP", "ON_RIM_ATTACK", "ON_POST_TOUCH", "ON_PASS_FIRST", "onball_isolation", "OFF_SPOTUP_SPACE", "OFF_CUT", "OFF_OFFSCREEN", "OFF_SCREEN_ROLE", "offball_crash_off_reb"]
    }
  }
}
```

- `locked`에 들어간 항목은 UI에서 비활성화하고, 고정 배지(`역할 고정`)를 노출한다.
- `selectable` 항목만 슬라이더/토글로 유저가 조정한다.
- 향후 역할 튜닝 시에는 `locked`만 바꾸면 UX 정책이 즉시 반영되도록 관리한다.
